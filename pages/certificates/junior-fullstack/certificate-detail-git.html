Global Information Tracker (GIT)

Belajar GIT Pemula (Codepolitan)

Belajar GIT Pemula (Codepolitan) adalah kelas online level pemula yang mengajarkan dasar-dasar penggunaan Git, yaitu sistem version control yang penting dalam pengembangan perangkat lunak untuk melacak perubahan kode dan kolaborasi tim. Materi mencakup pengenalan Git dan manfaatnya, cara menginstal Git, melakukan konfigurasi awal, membuat repository, serta dasar-dasar operasi seperti menyalin repository, mengirim perubahan (commit), dan menggunakan antarmuka Git GUI. Kelas ini disusun secara langkah demi langkah agar cocok bagi yang baru pertama kali belajar Git untuk keperluan pengembangan proyek dan kolaborasi coding.

Belajar GIT Lanjutan (Codepolitan)

Belajar GIT Lanjutan (Codepolitan) adalah kelas online lanjutan yang dirancang untuk peserta yang sudah memahami dasar-dasar Git dan ingin memperdalam penggunaan version control ini dalam pengembangan proyek sehari-hari. Kursus ini memperluas kemampuan Git dengan konsep dan praktik yang lebih kompleks, termasuk manajemen riwayat perubahan (history), teknik reset, revert, dan restore, pemahaman branching dan merging, serta cara bekerja dengan repository online seperti GitHub untuk kolaborasi tim. Materi disusun secara bertahap agar peserta dapat meningkatkan keterampilan Git mereka dari sekadar operasi dasar ke penggunaan lanjutan dalam konteks kerja nyata pengembangan perangkat lunak.
