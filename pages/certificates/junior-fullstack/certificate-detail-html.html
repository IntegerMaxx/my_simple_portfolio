BROWSER, HTTP, DNS, AND HOSTING (MySkill)

Modul ini menjelaskan dasar cara kerja web dari sisi infrastruktur: peran browser saat meminta halaman, cara protokol HTTP mentransfer data, bagaimana DNS menerjemahkan nama domain ke alamat IP, serta opsi dasar hosting dan alur deployment sederhana. Cocok untuk pemula yang ingin paham gambaran end-to-end bagaimana sebuah halaman web sampai ke pengguna.

Apa yang dipelajari (ringkas): konsep browser → request/response HTTP (method, status), bagaimana DNS lookup bekerja, tipe hosting dasar (shared, VPS, cloud) dan gambaran proses deploy.

Internet Introduction (MySkill)

Materi pengantar internet yang membahas konsep fundamental jaringan dan ekosistem web — termasuk peran browser, protokol dasar, arsitektur client-server, dan istilah-istilah penting yang sering muncul saat bekerja di web development. Bagus sebagai penguat pemahaman sebelum masuk ke teknologi front-end.

Apa yang dipelajari (ringkas): definisi internet & web, model client-server, protokol dasar (HTTP/HTTPS), serta gambaran alur permintaan data dari titik awal sampai tampil di browser.

Tools Instalation (MySkill)

Modul instalasi alat (tools) memberi panduan praktis menyiapkan lingkungan pengembangan: instalasi editor, Node/NPM (jika diperlukan), Git, dan tool pendukung lain yang diperlukan untuk latihan praktikal di kursus Web/Frontend. Biasanya disertai petunjuk langkah demi langkah dan troubleshooting dasar.

Apa yang dipelajari (ringkas): cara download & install tool, konfigurasi awal (PATH, credential Git), serta tips verifikasi instalasi agar siap mengerjakan materi berikutnya. (Catatan: MySkill menyediakan beberapa materi instalasi pada course/bootcamp terkait; halaman Introduction and Installation dan FAQ bootcamp memberi contoh pola tersebut).

HTML Introduction (MySkill)

Pengenalan HTML yang mengajarkan struktur dokumen HTML, tag dasar (head, body, heading, paragraph), serta cara menulis markup yang semantic dan rapi. Materi ini biasanya disusun dalam beberapa video singkat dengan contoh langsung pembuatan halaman sederhana.

Apa yang dipelajari (ringkas): struktur dokumen HTML, tag-tag dasar, membuat layout sederhana dengan elemen HTML, dan best practice penulisan markup. Modul ini masuk dalam topik Frontend — Introduction to HTML di MySkill.

HTML Attributes & Block Elements (MySkill)

Fokus pada atribut HTML (seperti id, class, src, href, alt, dsb.) dan perbedaan antara block-level dan inline elements. Modul ini membantu kamu paham kapan menggunakan atribut tertentu dan bagaimana elemen memengaruhi flow/layout halaman.

Apa yang dipelajari (ringkas): cara kerja atribut umum, perbedaan block vs inline, contoh penggunaan praktis (mis. kapan gunakan <div> vs <span>), serta implikasi terhadap styling dan layout.

HTML Images & Formatted Text (MySkill)

Materi ini membahas cara menampilkan gambar serta memformat teks di dalam halaman HTML. Peserta diperkenalkan pada penggunaan elemen <img> beserta atribut pentingnya, serta berbagai tag HTML untuk memformat teks agar lebih informatif dan mudah dibaca. Modul ini biasanya disajikan dengan contoh langsung penerapan gambar dan teks pada halaman profil atau halaman konten sederhana.

Apa yang dipelajari (ringkas): penggunaan tag <img> dan atributnya, formatted text seperti bold dan italic, penulisan teks yang informatif, serta penerapan gambar dan teks dalam struktur HTML dasar. Modul ini termasuk dalam topik Frontend — HTML di MySkill.

Inline Formatting Text (MySkill)

Modul ini berfokus pada pemformatan teks secara inline tanpa mengganggu alur paragraf. Peserta mempelajari berbagai tag HTML yang digunakan untuk memberi penekanan atau makna pada teks tertentu di dalam satu baris. Materi ini penting untuk membuat konten teks lebih terstruktur, jelas, dan semantik.

Apa yang dipelajari (ringkas): penggunaan elemen inline seperti <strong>, <em>, <b>, <i>, <small>, dan <mark>, perbedaan penekanan visual dan semantik, serta penerapan formatting teks dalam paragraf HTML. Modul ini termasuk dalam topik Frontend — HTML di MySkill.

Semantic HTML & Generic Element (MySkill)

Materi ini mengenalkan konsep HTML semantik dan perbedaannya dengan elemen generik. Peserta belajar bagaimana menyusun struktur halaman web menggunakan elemen yang memiliki makna jelas, sehingga kode lebih mudah dibaca, di-maintain, dan ramah terhadap mesin pencari serta aksesibilitas.

Apa yang dipelajari (ringkas): penggunaan elemen semantik seperti <header>, <main>, <section>, <article>, dan <footer>, fungsi elemen generik <div> dan <span>, serta best practice penulisan struktur HTML yang semantik. Modul ini termasuk dalam topik Frontend — HTML di MySkill.

Table In Profile Page (MySkill)

Modul ini membahas penggunaan tabel HTML untuk menampilkan data terstruktur pada halaman profil. Peserta mempelajari bagaimana menyusun informasi seperti biodata, riwayat pendidikan, atau pengalaman kerja dalam bentuk tabel agar rapi dan mudah dibaca.

Apa yang dipelajari (ringkas): penggunaan elemen <table>, <tr>, <th>, dan <td>, penyusunan data dalam baris dan kolom, serta penerapan tabel pada halaman profil sederhana. Modul ini termasuk dalam topik Frontend — HTML di MySkill.

Forms In Profile Page (MySkill)

Materi ini memperkenalkan pembuatan form HTML yang umum digunakan pada halaman profil atau halaman kontak. Peserta belajar bagaimana membuat form untuk menerima input dari pengguna dengan struktur yang rapi dan mudah digunakan.

Apa yang dipelajari (ringkas): penggunaan elemen <form>, <input>, <textarea>, <select>, dan <button>, penerapan atribut form dasar, serta pembuatan form sederhana pada halaman profil. Modul ini termasuk dalam topik Frontend — HTML di MySkill.

HTML Style & Colors (MySkill)

Materi ini menjelaskan cara mengatur tampilan dan warna elemen HTML agar halaman web terlihat lebih menarik dan sesuai desain. Peserta belajar berbagai cara memberi style pada teks dan elemen HTML menggunakan atribut style langsung di HTML (inline style) serta konsep dasar warna dan visual yang sering digunakan dalam pembuatan antarmuka halaman

Apa yang dipelajari (ringkas): cara menerapkan style dan warna pada elemen HTML, penggunaan atribut style, pengaturan warna teks dan latar belakang, serta praktik styling sederhana untuk membuat halaman lebih estetis. Materi ini merupakan bagian dari topik Frontend — HTML di MySkill.

HTML Classes & ID (MySkill)

Modul ini membahas penggunaan atribut class dan id dalam elemen HTML. Kedua atribut ini digunakan untuk memberi identitas pada elemen sehingga dapat diatur tampilannya dengan CSS atau diakses melalui JavaScript. Pemahaman tentang class dan id penting untuk struktur halaman yang terorganisir dan styling yang efektif

Apa yang dipelajari (ringkas): pengenalan atribut class dan id, cara menggunakannya dalam penulisan HTML, perbedaan keduanya, serta bagaimana mereka membantu dalam pengelompokan dan penargetan elemen untuk styling atau scripting. Topik ini ada dalam rangkaian Frontend — HTML di MySkill.

HTML Media (MySkill)

Materi ini memperkenalkan cara menambahkan elemen media seperti gambar, video, atau audio ke halaman HTML. Peserta belajar tag-tag HTML yang digunakan untuk menyisipkan media dan bagaimana atributnya mempengaruhi tampilan serta perilaku media di browser.

Apa yang dipelajari (ringkas): cara menampilkan media (gambar, video, audio) di HTML, penggunaan atribut masing-masing elemen media, serta praktik memasukkan media ke dalam struktur halaman web sederhana dalam modul Frontend — HTML di MySkill.

Practice: Making A "To Do List" Website (MySkill)

Fokusnya adalah menggabungkan berbagai elemen HTML seperti teks, input, tombol, serta struktur halaman secara keseluruhan dalam sebuah proyek nyata yang bisa menjadi portofolio.

membuat halaman web fungsional sederhana (“To Do List”) menggunakan HTML, menerapkan struktur dan atribut HTML yang telah dipelajari sebelumnya, serta mempraktikkan penggunaan elemen form, teks, dan penataan halaman dalam konteks proyek nyata. Modul ini termasuk Portfolio Project di bagian Frontend — HTML MySkill.
