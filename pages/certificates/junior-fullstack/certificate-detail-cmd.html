Command Prompt

Belajar Menggunakan Terminal CMD (Codepolitan)

Belajar Menggunakan Terminal CMD (Codepolitan) adalah kelas online level pemula yang mengajarkan cara menggunakan terminal atau Command Prompt (CMD) untuk kebutuhan development, khususnya dalam pengembangan web. Materi kursus mencakup pengenalan alasan pentingnya menguasai terminal, cara menginstal dan menggunakan terminal di berbagai sistem operasi (Windows, macOS, Linux), serta perintah-perintah dasar seperti menampilkan isi direktori (ls/dir), navigasi folder (cd), pembuatan folder (mkdir), pembuatan file (touch), dan penghapusan file/folder (rm). Tujuan kelas ini adalah agar peserta dapat mengoperasikan terminal secara efektif dalam pengelolaan file dan proyek saat bekerja dengan alat-alat pengembangan modern.
