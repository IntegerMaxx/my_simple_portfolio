# Portfolio Website (Static)

## Deskripsi

Website ini adalah **portfolio statis** yang dibuat untuk menampilkan profil, keahlian, dan karya secara sederhana, cepat, dan modern. Website tidak menggunakan backend atau database, sehingga dapat dihosting di layanan static hosting seperti **GitHub Pages, Netlify, atau Vercel**.

Fokus utama website ini adalah:

* Tampilan **bersih dan flat**
* Animasi **halus (smooth)**
* Performa ringan
* Mudah dipelajari dan dikembangkan

---

## Teknologi yang Digunakan

* **HTML5** – struktur halaman
* **CSS3** – layout, animasi, dan styling
* **JavaScript (Vanilla)** – interaksi dan efek scroll

> Tidak menggunakan framework (React, Vue, dll) agar tetap ringan dan mudah dipahami.

---

## Fitur Utama

* Responsive design (mobile & desktop)
* Smooth scrolling
* Animasi berbasis scroll
* Navigasi sederhana
* Struktur kode rapi dan mudah dikembangkan

---

## Cara Menjalankan

Karena ini adalah website statis, Anda bisa menjalankannya dengan sangat mudah:

### 1. Jalankan Langsung

Cukup buka file berikut di browser:

```text
index.html
```

### 2. Menggunakan Live Server (Opsional)

Disarankan menggunakan **Live Server** agar lebih nyaman saat development:

* VS Code Extension: *Live Server*
* Klik kanan `index.html` → **Open with Live Server**

---

## Hosting

Website ini dapat dihosting secara gratis menggunakan:

* GitHub Pages
* Netlify
* Vercel

### Contoh GitHub Pages

1. Push project ke repository GitHub
2. Masuk ke **Settings → Pages**
3. Pilih branch `main`
4. Website akan tersedia secara publik
