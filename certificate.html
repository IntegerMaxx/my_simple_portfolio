Junior FullStack

HTML

HTML (HyperText Markup Language) adalah bahasa markup standar yang digunakan untuk membangun struktur dan konten dasar sebuah halaman web. HTML berfungsi sebagai kerangka utama yang menyusun elemen-elemen seperti teks, gambar, tautan, tabel, dan formulir agar dapat ditampilkan dengan benar di browser. Materi HTML bertujuan untuk membantu pembelajar memahami cara menyusun halaman web secara terstruktur, konsisten, dan mudah dipahami.

CSS

CSS (Cascading Style Sheets) adalah bahasa stylesheet yang digunakan untuk mengatur tampilan dan tata letak halaman web yang ditulis dengan HTML. CSS berfungsi memisahkan antara struktur konten (HTML) dan presentasi visual, sehingga halaman web menjadi lebih rapi, konsisten, dan mudah dikelola.

JavaScript

JavaScript adalah bahasa pemrograman tingkat tinggi yang digunakan untuk membuat halaman web menjadi interaktif, dinamis, dan lebih hidup. Pada awal kemunculannya tahun 1995, JavaScript diciptakan oleh Brendan Eich di Netscape hanya dalam waktu 10 hari sebagai bahasa skrip yang berjalan di browser untuk memberikan interaksi pada elemen-elemen HTML. Namun seiring berkembangnya teknologi web, JavaScript telah berevolusi menjadi salah satu bahasa pemrograman paling populer di dunia dan menjadi tulang punggung dalam pengembangan aplikasi modern.

Flutter

Flutter adalah sebuah framework UI open-source yang dikembangkan oleh Google untuk membangun aplikasi lintas platform (cross-platform) dengan satu basis kode (single codebase). Dengan Flutter, developer dapat membuat aplikasi yang berjalan secara native di Android, iOS, Web, Windows, macOS, hingga Linux, tanpa harus menulis kode yang berbeda untuk setiap platform.

Program Vocational School Graduate Academy (VSGA) Junior Web Developer merupakan pelatihan dan sertifikasi kompetensi yang berfokus pada pengembangan keahlian dalam membangun website dan aplikasi web tingkat dasar. Peserta dibekali kemampuan dalam pemrograman web menggunakan teknologi standar industri, seperti HTML, CSS, JavaScript, serta pengelolaan database dan dasar-dasar backend. Python

Python adalah bahasa pemrograman tingkat tinggi yang dirancang dengan sintaks sederhana, mudah dibaca, dan mudah dipelajari. Python menekankan keterbacaan kode, sehingga sangat cocok untuk pemula sekaligus tetap kuat dan fleksibel untuk pengembang berpengalaman.

Dart

Dart adalah bahasa pemrograman modern yang dikembangkan oleh Google dan dirancang untuk membangun aplikasi dengan performa tinggi dan antarmuka yang konsisten. Dart dikenal dengan sintaks yang bersih, terstruktur, dan mudah dipahami, terutama bagi pengembang yang sudah familiar dengan bahasa berorientasi objek.

AWS Cloud

AWS Cloud (Amazon Web Services) adalah platform komputasi awan (cloud computing) yang menyediakan berbagai layanan teknologi informasi melalui internet. AWS memungkinkan individu maupun perusahaan untuk menggunakan sumber daya komputasi tanpa harus memiliki dan mengelola infrastruktur fisik sendiri seperti server, jaringan, dan pusat data.

SQL

SQL (Structured Query Language) adalah bahasa standar yang digunakan untuk mengelola dan mengolah data pada basis data relasional. SQL memungkinkan pengguna untuk berinteraksi dengan database, seperti mengambil, menambahkan, mengubah, dan menghapus data secara terstruktur dan efisien.

Algoritma

Algoritma adalah serangkaian langkah logis dan sistematis yang disusun untuk menyelesaikan suatu masalah atau mencapai tujuan tertentu. Algoritma menjadi dasar utama dalam pemrograman dan ilmu komputer, karena setiap program pada dasarnya merupakan implementasi dari satu atau lebih algoritma.

Algoritma & Pemrograman Dasar

Algoritma & Pemrograman Dasar merupakan fondasi utama dalam dunia pemrograman dan pengembangan perangkat lunak. Materi ini membahas cara berpikir logis dan sistematis untuk menyelesaikan masalah, serta bagaimana solusi tersebut diterjemahkan ke dalam bentuk kode program.

Mengenal Pemrograman Komputer

Mengenal Pemrograman Komputer adalah materi pengantar yang bertujuan memberikan pemahaman dasar tentang apa itu pemrograman dan bagaimana komputer bekerja melalui instruksi-instruksi yang diberikan manusia. Pemrograman komputer merupakan proses menulis, menguji, dan menjalankan perintah agar komputer dapat menyelesaikan suatu tugas atau memecahkan masalah tertentu.

Command Prompt

Command Prompt adalah antarmuka baris perintah (Command Line Interface/CLI) pada sistem operasi Windows yang digunakan untuk berinteraksi langsung dengan komputer melalui perintah teks. Dengan Command Prompt, pengguna dapat menjalankan berbagai instruksi tanpa menggunakan tampilan grafis (GUI).

GIT (Global Information Tracker)

Git adalah sistem kontrol versi terdistribusi (Distributed Version Control System) yang digunakan untuk mengelola dan melacak perubahan pada kode sumber atau file proyek. Git membantu pengembang mencatat setiap perubahan yang dilakukan, sehingga riwayat pengembangan proyek dapat dilihat, dikelola, dan dikembalikan ke versi sebelumnya jika diperlukan.

Dasar Visualisasi Data

Dasar Visualisasi Data adalah pengenalan terhadap konsep, prinsip, dan teknik untuk menyajikan data dalam bentuk visual agar lebih mudah dipahami, dianalisis, dan dikomunikasikan. Visualisasi data membantu manusia menangkap pola, tren, perbandingan, serta anomali yang sulit terlihat jika hanya disajikan dalam bentuk angka atau tabel.

Machine Learning

Machine Learning adalah cabang dari kecerdasan buatan (Artificial Intelligence) yang berfokus pada pengembangan sistem atau algoritma yang memungkinkan komputer belajar dari data dan pengalaman tanpa harus diprogram secara eksplisit untuk setiap tugas. Dengan machine learning, sistem dapat mengenali pola, membuat prediksi, dan mengambil keputusan berdasarkan data yang tersedia.

Analisis Data

Analisis Data adalah proses sistematis untuk mengumpulkan, mengolah, mengeksplorasi, dan menafsirkan data guna memperoleh informasi, pola, dan insight yang bermakna. Tujuan utama analisis data adalah membantu memahami kondisi yang terjadi, menjawab pertanyaan tertentu, serta mendukung pengambilan keputusan yang lebih tepat dan berbasis fakta.

Dasar AI

Dasar AI (Artificial Intelligence) adalah pengenalan terhadap konsep, prinsip, dan pendekatan utama dalam pengembangan sistem komputer yang mampu meniru atau mendukung kecerdasan manusia. AI berfokus pada bagaimana mesin dapat melakukan tugas yang biasanya membutuhkan kemampuan berpikir manusia, seperti memahami informasi, belajar dari pengalaman, mengenali pola, dan mengambil keputusan.

Command Linux

Command Linux adalah kumpulan perintah dasar yang digunakan untuk berinteraksi dengan sistem operasi Linux melalui antarmuka baris perintah (command line interface atau terminal). Perintah-perintah ini memungkinkan pengguna mengelola sistem, file, direktori, proses, dan konfigurasi secara langsung dan efisien tanpa menggunakan antarmuka grafis.

Jaringan Komputer

Jaringan Komputer adalah sistem yang menghubungkan dua atau lebih perangkat komputer agar dapat saling berkomunikasi, bertukar data, dan berbagi sumber daya seperti file, printer, dan koneksi internet. Jaringan memungkinkan informasi dikirim dari satu perangkat ke perangkat lain secara cepat dan terstruktur melalui media kabel maupun nirkabel.

Cyber Security

Cyber Security adalah bidang yang berfokus pada perlindungan sistem komputer, jaringan, aplikasi, dan data dari ancaman digital seperti peretasan, malware, pencurian data, dan penyalahgunaan sistem. Tujuan utama cyber security adalah menjaga kerahasiaan, integritas, dan ketersediaan informasi agar tetap aman dari akses atau gangguan yang tidak sah.
